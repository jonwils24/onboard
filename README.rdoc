== Welcome to OnBoard

A stock Rails application


== What's included?

1. Authentication with 
* Twitter
* Facebook
* Email/Password

== To configure

You will need to set up your environment to contain the following constancts.

	export GMO_TWITTER_CONSUMER_KEY='your-key' 
	export GMO_TWITTER_CONSUMER_SECRET='your-secret'
